Files: files ending with .meanQ
Description: output from fastSTRUCTURE analysis after 10 runs testing 2-10 K groups. Used to make plots using poppr. 

Files are named: test(run).(K group).meanQ
e.g test10.9 refers to run=10 and K group=9. 
Files for run 1 do not indicate the run number (e.g test.2.meanQ indicates run=1, K group=2)
