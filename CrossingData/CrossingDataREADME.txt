Crossing data analyses

Statistical analysis on crossing data to examine whether there is a crossing incompatibility
in the form of barrier to seed set between Ipomoea "austinii" and Ipomoea grandifolia and 
and between other species

Original dataset:
RIseeds_raw_clean_data.xlsx

Cleaned seed data set used for all analyses:
RIseeds_final.txt

For G-test analyses on fruit set:
SeedPercent_Gtest.txt

Link to Ipo_RI_survey.csv:
https://doi.org/10.5061/dryad.2bvq83bnx

R file for data analyses:
SumSatasRI_final.R
